# Корпоративный RAG-ассистент
Этот репозиторий содержит материалы проекта по созданию воспроизводимой MLOps-системы для LLM-ассистента на базе технологии RAG . Проект реализует полный цикл эксплуатации ML-модели

Проект реорганизован для удобной интеграции в CI/CD: код разделен на изолированные этапы с четкими входами/выходами, что позволяет запускать каждый шаг независимо в автоматизированных пайплайнах.

**Текущий этап:** формирование базы знаний. Пайплайн включает три последовательных шага: `download_data.py` загружает и очищает .md/.pdf-файлы, конвертируя их в единый формат; `splitter.py` разбивает текст на чанки с учетом структуры и готовит train/val-выборки; `evaluation.py` тестирует RAG-систему на основе ChromaDB и локальной LLM (Ollama), оценивая качество ответов. Все эксперименты логируются в MLflow — метрики и результаты доступны через веб-интерфейс для анализа и сравнения версий.

## Инструкция по запуску
1. Склонируйте репозиторий и в терминале перейдите в папку `mlops_project`
    ```bash
    git clone https://github.com/bitterrch/mlops_project.git
    cd mlops_project
2. Создайте и активируйте виртуальное окружение
    ```bash
    python -m venv venv
    venv\Scripts\activate
3. Установите зависимости 
    ```bash
    pip install -r requirements.txt
4. Создайте ядро для Jupyter Notebook
    ```bash
    python -m ipykernel install --user --name mlops_project
5. Чтобы запустить процесс парсинга сайта и наполнить папку `data/school_knowledge_base` свежими файлами, выполните
    ```bash
    python src/data/parser.py
6. Чтобы посмотреть результаты анализа текстов, запустите Jupyter:
    ```bash
    jupyter notebook
    ```
    Затем в Jupyter-е откройте `notebooks/01_eda.ipynb`, выберите ядро `mlops_project` и выполните весь код
7.  Установите Ollama и скачайте необходимые модели
    ```bash
    ollama pull qwen2.5:7b
    ollama pull llama3.2
    ```
8.  В одном терминале запустите интерфейс MLflow и потом откройте ссылку http://localhost:5000
    ```bash
    mlflow ui
    ```
9.  Запустите пайплайн, в который входит скачивание данных, сплиттер и оценка
    ```bash
    python -m src.pipeline
    ```
10. В другом терминале запустите веб-приложение
    ```bash
    streamlit run app.py
    ```
